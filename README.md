# Ayush Madhukar Pixel Bedroom

A Vite + TypeScript personal website for `ayumad.me`.

V1 is an original Game Boy Advance-style bedroom rendered on a 240x160 canvas and
scaled up with nearest-neighbor filtering. Visitors move around the room with
WASD or arrow keys, inspect objects with Z or Enter, and can also click or tap
objects directly.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The `public/CNAME` file is copied into the Vite build output for GitHub Pages
custom-domain deployment at `ayumad.me`.

## V1 Notes

- The terminal concept is intentionally paused while the room feel is refined.
- No CSS blob room, Three.js, Blender assets, or walkable 3D scene ships in V1.
- No copyrighted game sprites are imported; the current tiles and sprites are
  original canvas drawings.
- No backend, auth, model API, or real AI-ush chatbot ships in V1.
- Contact links and resume PDF links are placeholders until public URLs are confirmed.
